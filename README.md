# Garment POS System

A comprehensive Point of Sale system with multi-tenant architecture.

## Project Structure

- **`frontend/`**: Vite-powered React application for the user interface.
- **`backend/`**: Node.js/Express server handling multi-tenant MySQL databases.

## Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. Configure `.env`
4. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

## Features
- Multi-tenant database management
- Super Admin dashboard for client management
- Comprehensive POS functionalities (Billing, Purchase, Stock management)
